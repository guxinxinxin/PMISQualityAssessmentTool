Web-based Project Management Information System (PMIS) Quality Assessment Tool

The Web-Based PMIS Quality Assessment Tool is a decision-support system designed to help project managers evaluate and compare project management information systems based on intangible software quality metrics. Developed from a tailored quality model, the tool incorporates key factors such as reliability, usability, functionality, efficiency, interoperability, accuracy, and portability, each broken down into measurable attributes. By collecting expert input on the priority and influence of these attributes, the tool calculates weighted scores to objectively assess different PMIS platforms. This enables users to make informed, data-driven decisions aligned with their project needs and organizational goals.
