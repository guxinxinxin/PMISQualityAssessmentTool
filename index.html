Web-based Project Management Information System (PMIS) Quality Assessment Tool for Intangible Metrics

This decision-support tool helps project managers evaluate and select the best web-based Project Management Information System (PMIS) for their needs. By assessing software based on key quality attributes like reliability, usability, and functionality, the system calculates a weighted score for different platforms. This provides project managers with an objective, data-driven basis to make informed decisions, ensuring their chosen PMIS aligns with specific project and organizational goals.

Please rate each attribute based on your experience using the 5-point Likert Scale.

1 = Very Poor, 2 = Poor, 3 = Average, 4 = Good, 5 = Excellent
